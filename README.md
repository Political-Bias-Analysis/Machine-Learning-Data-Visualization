# ML
The ML folders contain the code we use to feed the clean Twitter and article data into the VADER sentiment analysis model. 

# Statistical Model/Analysis_Nancy
These folders contains the code we use to fit the two Linear Regression models. One on article sentiment and Normalized Vote Count. And the other is the Twitter sentiment and Normalized Vote Count. 

# R_Analysis_Deb
This folder contains the code we use to build the model to compare the sentiment of the article data and the Twitter data.
